GUI APPLICATION
----------------

1) Run the given command in the terminal 

i) IP=$(ifconfig en0 | grep inet | awk '$1=="inet" {print $2}')
ii) echo $IP
iii) xhost + $IP

(or)

ifconfig en0

2) start Xquartz Application on your computer 

References:

i) https://sourabhbajaj.com/blog/2017/02/07/gui-applications-docker-mac/
